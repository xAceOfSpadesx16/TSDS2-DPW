Descripción del Proyecto

Este portfolio web fue desarrollado como trabajo integrador final del segundo cuatrimestre de la materia "Diseño de Páginas Web". El proyecto reúne todas las actividades realizadas durante el cuatrimestre, organizadas en una estructura semántica coherente que simula un pequeño sitio web completo.

Estructura del Sitio Web

Páginas Incluidas:

  • index.html: Página principal con descripción general del portfolio
  • actividad1.html: Enlaces y formato de texto (Actividad 1)
  • actividad2.html: Listas, imágenes y contenido multimedia (Actividad 2)
  • actividad3.html: Tablas y formularios (Actividad 3)
  • detalles.html: Esta página con información técnica del proyecto

Implementación de Etiquetas Semánticas HTML5

Estructura Principal:

  • <header>: Encabezado general del sitio presente en todas las páginas
  • <nav>: Menú de navegación que conecta todas las páginas
  • <main>: Contenido principal de cada página
  • <footer>: Pie de página con información de copyright

Estructura de Contenido:

  • <section>: Secciones temáticas dentro del contenido
  • <article>: Contenido independiente y autocontenido
  • <aside>: Información complementaria
  • <figure>: Contenido multimedia con descripción

Contenido de las Actividades

Actividad 1 - Enlaces y Formato:

  • Estructura básica de HTML5
  • Etiquetas de formato: <em>, <strong>, <mark>, <del> <s>
  • Tres tipos de hipervínculos: interno, externo, entre páginas
  • Tres atributos diferentes: title, target="_blank", accesskey

Actividad 2 - Multimedia:

  • Listas ordenadas, desordenadas y de definición
  • Lista de términos con imágenes representativas
  • Dos archivos de audio con múltiples fuentes
  • Video con subtítulos en formato .vtt
  • Galería de imágenes adaptable con <picture>
  • Contenido embebido: YouTube, Google Maps, PDF

Actividad 3 - Tablas y Formularios:

  • Dos tablas con estructura semántica completa
  • Uso de <thead>, <tbody>, <tfoot>
  • Etiquetas <caption> para contexto
  • colspan y rowspan correctamente implementados
  • Celdas de cabecera con atributo scope
  • Formulario completo con múltiples tipos de inputs
  • Elementos de formulario avanzados y validación

Características Técnicas Implementadas

  • Accesibilidad: Atributos aria-current, labels asociados, estructura semántica
  • Responsive Design: Meta viewport, imágenes adaptables
  • SEO: Meta description, title descriptivo
  • Compatibilidad: Múltiples fuentes de audio/video, imágenes, etc
  • Navegación: Menú en todas las páginas, enlaces internos y externos
  • Validación: Formulario con atributos required, type, pattern, multiple, size, etc

Estilado y Diseño

Framework CSS:

El proyecto utiliza Bootstrap 5.3.2 como framework CSS principal para la estructura, los componentes y la responsividad del sitio.

Sistemas de Layout:

Se implementaron dos sistemas de layout principales:

  • Flexbox: Utilizado extensivamente para el layout general del sitio:
    • Estructura principal: d-flex flex-column en el body para mantener el footer fijo al final
    • Contenido principal: flex-grow-1 en el main para ocupar el espacio disponible
    • Navegación: uso de flexbox para alinear elementos del menú
    • Elementos de contenido: alineación y distribución de elementos (listas, botones, etc.)
  • Grid System: Utilizado principalmente en la Actividad 2:
    • Galería de imágenes: grid responsivo con row-cols-1 row-cols-md-2 row-cols-lg-3
    • Sección de audios: distribución en columnas con row y col-md-6
    • Contenido embebido: organización de iframes con sistema de grid
    • Formulario de Actividad 3: d-grid para los botones del formulario

Estilos Personalizados:

Se implementaron estilos personalizados en styles.css que complementan Bootstrap, incluyendo:

  • Variables CSS para la paleta de colores personalizada
  • Clases de color de fondo y texto (bg-pastel-*, text-pastel-*)
  • Estilos personalizados para botones (btn-pastel-*, btn-outline-pastel-*)
  • Efectos de transición en hover (hover-transition)

Paleta de Colores:

Se utilizó una paleta de colores pastel armoniosa basada en tonos azules y verde agua:

  • Primary: #B8D4E3 - Azul cielo suave (Color principal)
  • Secondary: #D4E8F0 - Azul cielo muy claro (Color secundario)
  • Accent: #C5E3D8 - Verde agua suave (Color de acento)
  • Dark: #A8C8D8 - Azul cielo medio (Color oscuro)

JavaScript Implementado

Evento de Formulario:

Se implementó un script JavaScript (form.js) que maneja el evento de envío del formulario en la Actividad 3:

  • Evento utilizado: submit del formulario
  • Funcionalidad: Previene el envío predeterminado del formulario y captura todos los datos ingresados
  • Método: Utiliza FormData para obtener todos los valores del formulario
  • Procesamiento: Convierte los datos a un objeto JavaScript y los muestra mediante alert()
  • Manejo de múltiples valores: Agrupa correctamente valores múltiples (como selects múltiples) en arrays
  • Propósito educativo: Demostrar la captura y visualización de datos de formularios en JavaScript
